# 三视图生成提示词模板

## 输入
- 人物照片

## 输出
- Q版卡通人物三视图（正面/侧面/背面并排在同一张图中）

## 提示词

严格遵守以下所有强制要求生成图片：

1. 必须完整生成Q版人物的正视图、侧视图、背视图三个视图，严格按「正视图→侧视图→背视图」的顺序水平并排排列在同一张图中，三个视图缺一不可，绝对不得仅生成正视图；

2. 基于上传的参考照片的人物面部、发型等核心特征，自动补全完整的全身下半身，生成完整的3头身Q版全身人物，不得截断身体，必须包含完整的腿部和鞋子；

3. 禁止复刻参考照片中的任何手势、动作，所有三个视图中人物的双手必须自然下垂、放在身体两侧，手臂放松，手掌自然张开，不得有其他任何姿势。

参考上传的个人形象照，保留人物的核心特征（比如：圆框眼镜、棕色波浪卷发、圆脸等），生成3头身Q版卡通全身人物的完整三视图，彩铅手绘风格，铅笔草稿线稿打底，彩铅上色，素描纸纹理质感，软萌可爱，圆脸蛋大眼睛，暖色调，柔和光影，米白色干净背景，专业角色设定稿，在每个视图下方清晰标注「正视图」「侧视图」「背视图」文字，手绘感强，治愈系插画，三个视图的人物造型、发型、服装细节完全一致，无差异

## 技术参数
- 模型：wan2.7-image
- 尺寸：16:9
- 输出文件：workspace/ip_triple_view.png
